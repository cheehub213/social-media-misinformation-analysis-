ğŸ“‹ Dimension Tables Description

1. Dim_Date (Date Dimension)

Column Data Type Description
Date_Key INT (PK) Surrogate key (format: YYYYMMDD)
Full_Date DATE Complete date value
Day INT Day of month (1-31)
Day_Name VARCHAR(10) Name of day (Monday, Tuesday, etc.)
Week_Number INT Week of year (1-52)
Month INT Month number (1-12)
Month_Name VARCHAR(15) Name of month
Quarter INT Quarter of year (1-4)
Year INT 4-digit year
Is_Weekend BOOLEAN TRUE if Saturday or Sunday
Is_Holiday BOOLEAN TRUE if public holiday

2. Dim_Source (Source Dimension)

Column Data Type Description
Source_Key INT (PK) Surrogate key
Source_Name VARCHAR(200) Name of the claim source
Source_Type VARCHAR(50) Type: Person, Organization, Media
Source_Category VARCHAR(50) Category: Politician, Celebrity, Media Outlet, Social Media
Is_Political_Figure BOOLEAN TRUE if political figure
Is_Media_Outlet BOOLEAN TRUE if media organization
Party_Affiliation VARCHAR(50) Political party if applicable

3. Dim_Label (Fact-Check Label Dimension)

Column Data Type Description
Label_Key INT (PK) Surrogate key
Label_Name VARCHAR(20) Official PolitiFact label
Label_Category VARCHAR(20) Grouped category: True, Partially True, False
Truth_Score INT Numeric truth score (0-5)
Label_Description VARCHAR(200) Detailed label meaning
Is_True BOOLEAN TRUE if claim is verified as true
Is_False BOOLEAN TRUE if claim is verified as false

ğŸ“Š Fact Table Description

FACT_CLAIMS (Central Fact Table)

Column Data Type Description
Claim_Key INT (PK) Surrogate key for each fact-check
Date_Key INT (FK) Publication date reference
Stated_Date_Key INT (FK) Date claim was made
Source_Key INT (FK) Reference to claim source
Label_Key INT (FK) Reference to fact-check verdict
Claim_Count INT Always 1, for counting claims
Word_Count INT Number of words in headline
Days_To_Verify INT Days between statement and publication
News_Headline TEXT Degenerate dimension - claim text
Link_Of_News VARCHAR(500) Degenerate dimension - source URL

ğŸ—‚ï¸ Grain Definition

Grain: One row per fact-checked claim published by PolitiFact.

Each record in the fact table represents a single claim that has been: - Made by a specific source - On a specific date (Stated_On) - Fact-checked and published on another date (Date) - Assigned a truth rating (Label)
